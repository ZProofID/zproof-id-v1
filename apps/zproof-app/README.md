# SocketFi

SocketFi is a smart wallet platform built on Soroban (Stellar), enabling seamless Web3 interactions through smart accounts, social intent transactions, and flexible fee handling.

It provides users and developers with a unified interface to manage assets, approve transactions, and interact with decentralized applications in a more intuitive and secure way.

## 🚀 Overview

SocketFi introduces a product-first approach to smart wallets by combining:

- Smart contract wallet accounts
- Social intent transaction flows
- Flexible gas abstraction
- Unified dApp access

The platform is designed to move beyond traditional wallet interactions and provide a more connected, user-friendly Web3 experience.

## ✨ Key Features

### 🔐 Smart Wallet Accounts

---

## 🧠 How It Works

1. **Create or connect a wallet**  
   Users onboard into a smart contract wallet experience.

2. **Initiate or receive intent**  
   Transactions can originate from direct actions, dApps, or social flows.

3. **Review and approve**  
   Users verify transaction details before execution.

4. **Execute on-chain**  
   Approved transactions are processed securely via the blockchain.

---

## 🏗️ Tech Stack

- **Frontend:** React + TailwindCSS
- **Blockchain:** Soroban (Stellar)
- **Authentication:** Passkeys
- **Signing:** BLS-based multi-node signature aggregation
- **Storage:** IndexedDB (client-side)

---

## 🔐 Security

- No direct access to user private keys
- Passkey-based authentication for all actions
- Multi-node signature aggregation for transaction execution
- Optional social recovery with verification controls

---

## 🌍 Ecosystem Focus

SocketFi is designed not only as a wallet, but as a **smart wallet infrastructure layer** that:

- Enables better onboarding for users
- Improves interaction patterns across dApps
- Supports flexible transaction and fee models
- Expands access to Web3 for social-native users
